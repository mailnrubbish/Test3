The Sales Rep Dashboard allows you to create, access and manage Opportunities in your sales campaigns. From the Dashboard you can create new Opportunities and access current Opportunities, route or mark them as lost. The Dashboard is fully customisable. Filters can be added and columns can be moved and layouts saved for future use.


The screen is divided into two sections

Event Summary view (1):        


This section of the screen displays a list of all Events to which you have been assigned and the total number of Opportunities at their various sales stages

 

Note

You must be added as a Resource to an Event to be able to see the Event in the Sales Rep dashboard. If you cannot see an Event, then contact your Manager

Opportunity Summary view (2):        


This section displays a list of all Opportunities in the Event selected above at their various sales stages and provides access to the Opportunity management screen by clicking the Opty Number hyperlink. The default view is All  which displays all Opportunities at all sales stages.  You can filter the Opportunities  by sales stage by clicking on a sales stage link





Sales Rep Dashboard - Detailed View



Function Buttons 


  1. My/All

Toggle to display either Opportunities with your Sales Code (My) or all Opportunities with all Sales Codes (All)

  1. Report Drop down list

Displays a list of all saved report layouts

  1. Actions        

Opens a menu of a list of actions that can be used to manage the displays and data (see related help).

  1. Show Current Registrations        

Displays a list of registrations for the selected Campaign (see related help)

  1. Route        

Routes one or more Opportunities to another Campaign in the same event. (see related help)        

  1. Mark as Lost        

Changes the sales stage of one or more Opportunities to the sales stage Lost. (see related help)

  1. Refresh        

Refreshes the data in the Opportunity Summary view  

  1. New Opportunity

Creates a new Opportunity for the selected Event. (see related help)

  1. Reset        

Resets the column layout to their default settings
