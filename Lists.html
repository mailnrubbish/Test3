Lists


Related Help



Lists are created one hour after the Event Plan is created. They are populated automatically when matching records are created e.g. If a contact downloads a PDF then their details are added to the PDF Download list


Note: For new events it is likely that intially all lists will be empty until contacts that match a list are created
