Creating a contact


  1. Complete all fields as necessary: Note: First Name, Last Name and Account Name are mandatory fields
  2. Use the radio button to choose to copy the Account address details to the Contact. If you chose No then you will need to add contact address details manually
  3. Click the Account Name search icon  to search for the Account Note: Do not type directly in this field. You must open the search window 
  4. Select/Create the Account
  5. Click Create Contact to create the contact and return to the Contact Lookup screen
